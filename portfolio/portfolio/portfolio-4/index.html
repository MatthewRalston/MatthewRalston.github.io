NGS-CI | Next Gen. Sequencing Complexity Index

ngsci is a python package and an original ruby-gem that is no longer under development. It is a simple projectRelax, it"s just algebraah. that creates a metric measuring sequencing complexity on a scale of 0 - 100.

  • 0 : means 0 sequencing reads on that base
  • 100 : maximum complexity/coverage etc. of information at that base.

I wrote it as a capstone project as part of my grad school work on Illumina sequencing data. It was not included in my Masters thesis.

Think through this.

For the first read in a group of reads overlapping the base of interest under saturated conditions, the summed overlap is equal to the triangular number of L - 1: i.e. there can be a mximum of L or L-1 reads (idk) overlapping that base.

Okay? So the answer to sequencing complexity is not on a scale of 0-L.
